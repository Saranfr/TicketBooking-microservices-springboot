## Learning Outcome
This project was built as part of learning distributed microservices architecture using Spring Boot, Kafka, Keycloak, and API Gateway.  
The implementation was adapted and extended from an existing open-source project for educational purposes.

## Key Concepts Demonstrated
- Microservices architecture
- Service discovery
- API Gateway routing
- Event-driven communication (Kafka)
- Authentication with Keycloak
- Docker container orchestration
